Using Python version 3.10.7

Config your keys:
1) Go inside "global.json" file
2) Insert your public and priate keys

Start system :
1) open terminal in current folder location
2) if using vanilla python (not anaconda)
    a) if you don't have "virtualenv" library, in terminal type "pip install virtualenv"
    b) create virtual enviroment, in terminal type "python -m venv env"
    c) start virtual enviroment, in terminal type "source env/bin/activate"
3) then type in terminal "pip install -r requirements.txt" to install all python packages for the system
4) lastly type in terminal "python server.py" to run the system
